# Ecommerce Backend Template

This is a simple **Ecommerce backend template** built with **Go**.

---

## Run the Server

Make sure you have **Go** installed.

```bash
go run main.go
```

Or, if you are using the **Makefile**:

```bash
make run
```

---

## Available Make Commands

- **Run the server**
  ```bash
  make run
  ```

- **Build the binary**
  ```bash
  make build
  ```

- **Clean binaries**
  ```bash
  make clean
  ```

- **Show project info**
  ```bash
  make about
  ```

- **Show help**
  ```bash
  make help
  ```
